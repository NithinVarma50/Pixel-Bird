# Clumsy Bird

A retro-themed Flappy Bird clone built with MelonJS.

![](http://i.imgur.com/Slbvt65.png)

## About the Game

Clumsy Bird is a fun and addictive game where you control a bird that must navigate through pipes without crashing. The game features retro-style graphics and smooth gameplay, inspired by the classic Flappy Bird.

## Features

- Retro-themed design
- Simple one-touch controls
- Score tracking
- Sound effects
- Responsive design

## How to Play

- Press SPACE or click to make the bird fly
- Avoid the pipes
- Press 'M' to mute/unmute sound

## Running Locally

- Install [Node.js](http://nodejs.org/download/) and [Grunt](http://gruntjs.com/)
- Install the dependencies

```
npm install
```

Then run the local server:

```
grunt connect
```

Open your browser at `http://localhost:8001/`
